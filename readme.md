## API PROVAS - [NestJs]

## Installation

```bash
$ npm install
```

## Running the app local

```bash
# development
$ npm run start:dev
```

## Running Docker

```bash
# development -> docker-compose -f <name-file-docker-compose> up
$ docker-compose -f docker-compose_dev.yml up
```

## Test

```bash
# e2e tests
$ npm run test:e2e:docker
```
